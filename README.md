# SampleWorkbench

A FreeCAD workbench with two commands: Add Box Sketch and Add Sphere.

## Installation

1. Copy the `SampleWorkbench` folder to your FreeCAD Mod directory:
   - Windows: `%APPDATA%\FreeCAD\Mod\`
   - Linux: `~/.FreeCAD/Mod/`
   - macOS: `~/Library/Preferences/FreeCAD/Mod/`

2. Restart FreeCAD

3. The workbench should appear in the workbench selector

## Commands

### Add Box Sketch
Creates a new sketch with a rectangular shape (100x100 mm) in the XY plane.

### Add Sphere
Adds a sphere with a radius of 50 mm to the document.

## Usage

1. Select the Sample workbench from the workbench dropdown
2. Click on the toolbar buttons or use the menu items to execute the commands
